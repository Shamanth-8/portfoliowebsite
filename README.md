# 🌐 Personal Portfolio Website

This is my personal portfolio website built with **Streamlit**.  
It showcases my skills, projects, and experience in **Data Science, Machine Learning, AI, and Full-Stack Development**.

---

## 🚀 Features
- **About Me** section with professional summary
- **Skills** categorized into Programming, Data Science, Web Development, and Tools
- **Projects Showcase** with descriptions and GitHub links
- **Contact Information** for easy networking
- Responsive layout with expandable sections
